IPC SDK

    • 1.1. 功能概述
    • 1.2. 涂鸦智能摄像机能力支持

1.1. 功能概述

涂鸦智能摄像机 Android SDK 提供了与远端摄像机设备通讯的接口封装，加速应用开发过程，后面章节统一称为 IPC SDK 。主要包括了以下功能：

  • 预览摄像机实时采集的影像。
  • 播放摄像机存储卡中录制的视频。
  • 手机端录制摄像机采集的影像。
  • 播放摄像机传来的音频与摄像机设备通话。
  • 下发与接受摄像机指令功能。
  • 云存储功能。
  • 侦测报警消息。

1.2. 涂鸦智能摄像机能力支持

  • 设备配网
    • 快连模式（Smart Config）
    • 热点模式（AP 模式）
    • 二维码配网
  • P2P 视频能力
    • 实时视频播放
    • SD 卡视频回放
    • 视频裸流数据
  • 低功耗门铃
    • 电池管理
    • 休眠唤醒
    • 门铃呼叫
  • 云存储
    • 云存储服务购买
    • 事件云存储
    • 云视频播放
  • 报警消息
    • 消息列表
    • 音视频消息播放
  • 扩展能力点
    • 云台控制
    • 存储卡管理
    • 侦测报警
    • 其他

以上能力视设备厂商不同有部分差异。
